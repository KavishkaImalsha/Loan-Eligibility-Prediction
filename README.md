# 🚀 Logistic Regression Prediction System
A full-stack Machine Learning application that predicts outcomes using a Logistic Regression model built with Scikit-learn.
The project includes data preprocessing, feature engineering, model training, and deployment using FastAPI, with a React-based frontend for user interaction.

# 📌 Project Overview
This project demonstrates an end-to-end machine learning workflow:
    - Data preprocessing and cleaning
    - Feature engineering
    - Model training using Logistic Regression
    - Model evaluation
    - API development with FastAPI
    - Interactive frontend built with React
    - The trained model achieved 82% accuracy on the evaluation dataset.

# 🧠 Machine Learning Model
Algorithm Used: Logistic Regression (from Scikit-learn)
Steps performed:
    - Data preprocessing
    - Handling missing values
    - Feature engineering
    - Feature scaling
    - Model training
    - Model evaluation
Model Performance:
    -Accuracy: 82%

# 🚀 How to Run the Project
1️⃣ Clone the repository
    - git clone https://github.com/KavishkaImalsha/Loan-Eligibility-Prediction.git
    - cd Loan-Eligibility-Prediction
2️⃣ Backend Setup
    - pip install -r requirements.txt
    - uvicorn main:app --reload (Run Backend)
3️⃣ Frontend Setup
    - cd frontend
    - npm install
    - npm run dev
